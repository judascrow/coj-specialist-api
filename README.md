# Golang API Starter

##### Golang v.1.14

Install:

```
git clone https://github.com/judascrow/cojspcl-api.git
rm .git
rm go.mod
rm go.sum
go mod init <New Mudule>
git init
cp .env.example .env
```

## Features

- RESTful API
- Gin
- Gorm
- Gin-Swagger
- Log-Zap
- Gin-Jwt
- Casbin

## COMMAND

Migrate & Seed :

```
go run main.go create seed
```

Run :

```
go run main.go
```

## API ROUTER URI

```
Host: localhost:8000
Base Path: /api/v1
```

| ------------------ | ------ | ------------ | --------- | -------------------- |
